## 🧩 주요 알고리즘 분류

### 자료구조 (Data Structures)
- **스택 (Stack)** - 괄호 검증, 탑, 히스토그램
- **큐 (Queue)** - 카드2, 프린터 큐, 요세푸스
- **덱 (Deque)** - AC
- **우선순위 큐 (Priority Queue)** - 최소/최대 힙, 중앙값, 카드 정렬
- **해시맵 (HashMap)** - 듣보잡, 포켓몬 마스터

### 탐색 (Search)
- **이분 탐색 (Binary Search)** - 나무 자르기, 공유기 설치, 용액
- **완전 탐색 (Brute Force)** - N-Queen, 외판원 순회
- **BFS/DFS** - 바이러스, 안전 영역, 유기농 배추

### 동적 계획법 (Dynamic Programming)
- **1차원 DP** - 피보나치, 1로 만들기, 계단 오르기, 파도반 수열
- **2차원 DP** - LIS (가장 긴 증가하는 부분 수열)
- **메모이제이션** - 피보나치 수, 포도주 시식

### 그리디 (Greedy)
- **활동 선택** - 회의실 배정
- **최적화** - 동전 0, ATM, 잃어버린 괄호

### 정렬 (Sorting)
- **기본 정렬** - 수 정렬하기, 단어 정렬
- **계수 정렬** - 수 정렬하기 3
- **좌표 압축** - 좌표 압축

### 수학 (Mathematics)
- **정수론** - 에라토스테네스의 체, 골드바흐의 추측
- **조합론** - 패션왕 신해빈
- **기하학** - 신발끈 공식 (다각형 넓이)
- **분할 정복** - 하노이 탑, Z, 행렬 제곱

### 문자열 (String)
- **문자열 처리** - KMP, 패턴 매칭

### 기타
- **투 포인터 (Two Pointer)** - 과일 탕후루
- **분할 정복 (Divide & Conquer)** - 쿼드트리, 종이의 개수
- **백트래킹 (Backtracking)** - N-Queen, 스도쿠
- **누적 합 (Prefix Sum)** - 구간 합 구하기
